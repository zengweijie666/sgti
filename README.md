# 三国人格鉴定 🎭

一个类似 MBTI 的趣味心理测试，用三国人物来鉴定你的性格类型。

## 📖 项目简介

三国乱世，人人有病。通过 20 道精心设计的题目，4 分钟左右的时间，暴露你的本性，看看你最适合三国中的哪个人物。

### 测试维度

- **职场行为** - 面对工作场景的反应
- **人际关系** - 处理朋友和社交的方式
- **价值观** - 对成功、金钱、道德的看法
- **决策风格** - 做决定时的思考模式
- **生活态度** - 日常生活中的行为特征
- **压力应对** - 面对困难和挫折的方式

## 🎯 人格类型

测试包含 10 种人格类型：

| 类型 | 阵营 | 特点 |
|------|------|------|
| 曹操型 | 魏·霸道总裁风 | 结果导向，决策果断 |
| 刘备型 | 蜀·人缘天花板 | 天生关系型选手 |
| 诸葛亮型 | 蜀·完美主义晚期 | 事必躬亲，鞠躬尽瘁 |
| 司马懿型 | 魏·熬鹰大师 | 长期主义，善于隐忍 |
| 关羽型 | 蜀·孤勇者 | 能力顶级，脾气也顶级 |
| 吕布型 | 群雄·跳槽之王 | 能力天花板，忠诚度地板 |
| 刘禅型 | 蜀·终极躺王 | 此间乐，不思蜀 |
| 袁绍型 | 群雄·纠结之王 | 资源最多，决策最慢 |
| 周瑜型 | 吴·卷到吐血 | 既生瑜，何生亮 |
| 貂蝉型 | 群雄·工具人 | 所有人都需要我，但没有人在乎我 |

## ✨ 功能特性

- 📱 **移动端优先** - 完美适配手机浏览器
- 💾 **断点续测** - 支持本地存储，刷新不丢失进度
- 🎨 **精美 UI** - 深色主题，流畅动画

## 🚀 快速开始

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/zengweijie666/sgti.git
cd sgti
```

2. 启动本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

3. 访问 `http://localhost:8000`

### 在线体验

直接访问部署后的网址即可开始测试。

## 🎨 自定义配置

### 修改题目

编辑 `app.js` 中的 `questions` 数组：

```javascript
const questions = [
  {
    id: 1,
    text: '你的题目内容',
    options: [
      { text: '选项 A', type: 'caocao' },
      { text: '选项 B', type: 'liubei' },
      // ...
    ]
  }
];
```

### 修改结果

编辑 `app.js` 中的 `results` 对象，每个类型包含：
- `name` - 中文名称
- `english` - 英文名称
- `camp` - 阵营标签
- `quote` - 经典语录
- `description` - 人格描述
- `weakness` - 致命弱点
- `ending` - 最终结局
- `weapon` - 武器
- `bgm` - 背景音乐
- `shareQuote` - 分享金句
- `bgColor` - 背景色
- `emoji` - 表情符号
- `image` - 图片路径
